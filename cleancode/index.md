# cleancode sub-skill index

Per-language Clean Code guides, sourced live from submodules under
`repos/`. Each row is `<slug> — <what it covers>`. Open
`./<slug>/README.md` for the full guide.

- `javascript` — Clean Code concepts adapted for JavaScript (ES6+). Variables, Functions, Objects & Data Structures, Classes, SOLID, Testing, Concurrency, Error Handling, Formatting, Comments. Source: ryanmcdermott/clean-code-javascript.
- `typescript` — Clean Code adapted for TypeScript. Same chapters as JS plus type-specific guidance (Variables, Functions, Objects, Classes, SOLID, Testing, Concurrency, Error Handling, Formatting, Comments). Source: labs42io/clean-code-typescript.
- `python` — Clean Code adapted for Python (PEP-aware). Variables, Functions, Classes, and Don't-Repeat-Yourself (DRY), with runnable examples. Source: zedr/clean-code-python.
- `php` — Clean Code adapted for PHP. Variables, Comparison, Functions, Objects & Data Structures, Classes, SOLID, DRY. Source: piotrplenik/clean-code-php.
- `go` — "Clean Go Code" long-form article. Idiomatic Go: naming, functions, returning values, pointers/mutability, interfaces, error handling, and Go-flavored clean-architecture. Source: Pungyeon/clean-go-article.
