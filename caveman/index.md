# caveman sub-skill index

- `caveman` — Ultra-compressed communication mode. Cuts token usage ~75% by speaking like a caveman while keeping full technical accuracy. Supports intensity levels: lite, full, ultra, wenyan. Trigger: "caveman mode", "talk like caveman", "less tokens", "/caveman".
- `caveman-commit` — Ultra-compressed commit message generator. Cuts noise while preserving intent. Conventional Commits format, subject ≤50 chars, body only when "why" isn't obvious. Trigger: "write a commit", "commit message", "/caveman-commit".
- `caveman-compress` — Compress natural language memory files (CLAUDE.md, todos, preferences) into caveman format to save input tokens. Trigger: "/caveman-compress FILEPATH", "compress memory file".
- `caveman-help` — Quick-reference card for all caveman modes, skills, and commands. One-shot display. Trigger: "/caveman-help", "caveman help".
- `caveman-review` — Ultra-compressed code review comments. Each comment is one line: location, problem, fix. Trigger: "review this PR", "code review", "/caveman-review".
- `caveman-stats` — Show real token usage and estimated savings for the current session. Reads from Claude Code session log. Trigger: "/caveman-stats".
- `cavecrew` — Decision guide for delegating to caveman-style subagents. Tells when to spawn investigator, builder, or reviewer subagents instead of doing work inline. Trigger: "delegate to subagent", "use cavecrew".
